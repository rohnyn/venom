# Vision

## One-line version

An open-source mobile operating system built around a persistent personal operator rather than apps alone.

## Product definition

The phone should not just host AI apps. The OS itself should provide:

- system-wide memory
- policy-bounded delegation
- cross-app action
- durable user context
- model-agnostic intelligence
- extension points for new capabilities

## User promise

The device should feel like:

- your phone still works the way you expect
- routine work gets quietly handled for you
- the assistant understands context across the device
- you can inspect, constrain, and revoke what it does

## Non-goals

- a chat app with deeper permissions
- a novelty launcher theme
- autonomy without auditability
- a closed assistant that cannot be modified or self-hosted
