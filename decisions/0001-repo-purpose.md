# ADR 0001: Repo Purpose

## Status

Accepted

## Context

The project is still at the thesis, architecture, and landscape stage. We need a place to accumulate research and decisions without prematurely forcing everything into a product spec or codebase.

## Decision

This repo will initially store:

- synthesized product and architecture docs
- raw research notes and source links
- decision records
- diagrams and working notes

It is not yet a source repo for implementation code.

## Consequences

- we can iterate on the thinking without pretending the implementation plan is settled
- decision history remains visible
- research and polished beliefs remain separate
